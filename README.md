# 🎵 Spotify UI Clone

A simple **Spotify-inspired static web interface** built using **HTML5** and **CSS3**. This project was created to practice modern web layout techniques, Flexbox, and front-end design while replicating the look and feel of Spotify's desktop interface.

## 🚀 Live Demo

🔗 **Website:**
https://gayatri-kulkarni24.github.io/spotify_clone/

## 📂 GitHub Repository

🔗 https://github.com/gayatri-kulkarni24/spotify_clone

## ✨ Features

* Spotify-inspired user interface
* Sidebar navigation
* Sticky top navigation bar
* Music player section
* Album and playlist cards
* Clean and organized layout
* Built entirely with HTML and CSS
* Deployed using GitHub Pages

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Montserrat)
* Font Awesome Icons
* Git & GitHub
* GitHub Pages

## 📁 Project Structure

```text
spotify_clone/

```


## 💻 Run Locally

1. Clone the repository

```bash
git clone https://github.com/gayatri-kulkarni24/spotify_clone.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or dependencies are required.
